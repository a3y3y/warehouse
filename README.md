Система учета инвентаря (Склад):
Бэкенд: Item (название, артикул, количество на складе, цена).
Angular: Таблица товаров. Кнопки "+" и "-" возле каждого товара, которые мгновенно обновляют количество на бэкенде. 
Подсвечивать красным товары, которых осталось меньше 5 штук.

Для скачивания репозитория с субмодулями запустить команду:
git clone --recurse-submodules https://github.com/a3y3y/warehouse.git

Для запуска, в корне проекта ввести команду:
docker compose up

Приложение будет доступно в браузере по адресу:
http://localhost:3000/
